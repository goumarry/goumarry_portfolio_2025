Site Web de Boutique en Ligne avec PHP Symfony

Objectif : Développer un site web de boutique en ligne permettant la gestion et la vente d’articles.

Outils : Symfony ( PHP )

Compétences : Maîtrise de PHP et du framework Symfony. Compréhension des principes MVC. Gestion de bases de données relationnelles avec Doctrine.

Ce projet consistait à créer un site web de boutique en ligne en utilisant le framework PHP Symfony. Le site permettait de gérer et de vendre des articles en ligne, incluant des fonctionnalités telles que la gestion des produits, des utilisateurs et des commandes. L’architecture MVC de Symfony a été utilisée pour structurer l’application, facilitant la séparation des préoccupations et rendant le code plus maintenable.
