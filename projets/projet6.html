Application mobile « L’école des Loustiques »

Objectif : Développer une application mobile éducative pour aider les enfants à apprendre les tables de multiplication et suivre leurs résultats.

Outils : Android Studio ( Java )

Compétences : Développement mobile avec Android Studio. Gestion d'une base de données locale. Interfaces interactives adaptées aux enfants.

Dans ce projet, intitulé « L’école des Loustiques », nous avons développé une application mobile éducative utilisant Android Studio. L’application permet aux enfants d’apprendre et de pratiquer les tables de multiplication. Les utilisateurs peuvent créer un compte, se connecter et suivre leurs progrès grâce à une base de données locale. Chaque session de pratique est enregistrée, et les résultats sont stockés pour permettre un suivi personnalisé de l’apprentissage.
