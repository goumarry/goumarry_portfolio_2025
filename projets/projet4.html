Amélioration d’une Application de Recherche de Stage

Objectif : Améliorer une application existante de recherche de stage, incluant des interfaces web et mobile.

Outils : Symfony ( PHP ) et Android Studio ( Java )

Compétences : Développement backend avec Symfony. Développement mobile avec Android Studio. Gestion des interfaces utilisateur pour différents rôles (admin et étudiant).

Ce projet consistait à améliorer une application de recherche de stage déjà existante. L’application comportait une interface web gérée avec Symfony et une application mobile développée avec Android Studio. Les améliorations apportées visaient à enrichir les fonctionnalités et à optimiser l’expérience utilisateur. Côté admin, des fonctionnalités ont été ajoutées pour gérer les offres de stage, suivre l’état des candidatures, et administrer les utilisateurs. Pour les étudiants, des fonctionnalités telles que la recherche d’offres, la candidature en ligne, et le suivi des candidatures ont été améliorées.
