Développement d'un serveur REST en Node.js

Objectif : Implémentation rigoureuse d'une API RESTful documentée avec OpenAPI.

Outils : Node.js, Express, OpenAPI

Compétences : Spécification d'API, Node.js, Express, OpenAPI, HATEOAS, tests, documentation, client REST.

Création d'une API REST pour gérer des joueurs de foot. Le serveur est documenté avec OpenAPI et permet d'exécuter les opérations CRUD classiques sur des entités liées.
